Overleaf PPT Package

Main file to compile:
sars_presentation.tex

What is included:
- A standalone Beamer presentation for the SARS project
- An assets/ folder with the screenshots and diagrams used by the slides
- UML and implementation flow diagrams generated directly in LaTeX

Slide order included in the deck:
1. Title Slide
2. Table of Contents
3. Abstract
4. Problem Statement
5. Introduction
6. Existing System
7. Proposed System
8. Objectives
9. Requirements
10. Architecture Diagram
11. UML Diagrams
12. Modules
13. Implementation
14. Code Snippet
15. Results / Output
16. Advantages
17. Limitations
18. Conclusion
19. Future Scope
20. References
21. Thank You

How to use in Overleaf:
1. Upload this entire folder to Overleaf.
2. Set the main file to sars_presentation.tex.
3. Compile with pdfLaTeX.

Notes:
- The deck is based on the verified thesis content and available project screenshots.
- If you want to add more teacher screenshots, place them inside assets/ and update sars_presentation.tex.
- If your college requires a different title slide format, edit the title, author, institute, or date block in the main file.
