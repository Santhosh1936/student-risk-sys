Overleaf Report Package

Main file to compile:
sars_thesis_report.tex

Upload this entire folder to Overleaf.

Required contents included:
- sars_thesis_report.tex
- thesis/frontmatter/
- thesis/chapters/
- thesis/appendices/
- thesis/assets/

After upload in Overleaf:
1. Set the main file to sars_thesis_report.tex
2. Compile

Note:
The report file is separate from the research paper file.
You do not need sars_paper.tex for the thesis report.
